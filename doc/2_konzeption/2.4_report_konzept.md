# 2.4 Report Konzept

Wenn der Mailbox-Watcher die Mails einer Projekt-Mailbox 체berpr체ft,
wird ein Report generiert welcher die Mails pro Ordner anzeigt.

## 2.4.1 JSON Datenstruktur

[Hier](https://jsonapi.org/) wird beschrieben wie die JSON Datenstruktur f체r APIs aussehen muss.

Was muss in der JSON Datenstruktur vorhanden sein?

* Projektname
* Beschreibung
* Alerts
* Postf채cher
  * Name
  * Status
  * Alerts
  * Ordner
    * Name
    * Beschreibung
    * Anzahl Mails
    * Regex
    * Max Age
    * Alerts
    * Alert Mails
      * Subject
      * From
      * Received at
